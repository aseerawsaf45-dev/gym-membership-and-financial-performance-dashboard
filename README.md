# Gym Membership & Financial Performance Dashboard | Power BI

An interactive Power BI dashboard built to monitor gym business performance across memberships, revenue, expenses, profit trends, and monthly client activity. This project combines dashboard design with business-focused KPI tracking to support better operational and financial decision-making.

## Dashboard Preview

![Dashboard Preview](images/dashboard-overview.png)

## Project Objective

The goal of this project was to design a visually engaging and business-relevant Power BI dashboard for a gym or fitness center. The dashboard helps track operational performance, membership distribution, financial trends, and monthly client activity in a single reporting view.

## Key Metrics Tracked

- Total Clients
- Total Trainers
- Total Revenue
- Total Expense
- Profit Trend
- Membership Status
- Active vs Expired Memberships
- Monthly Client Trend

## Dashboard Features

- Interactive business dashboard built in Power BI
- Dark-themed modern UI design
- Financial trend visualization for revenue, expenses, and profit
- Membership monitoring by category
- Monthly client trend analysis
- Operational KPIs displayed through summary cards
- Portfolio-ready layout for business intelligence presentation

## Key Insights

- Platinum memberships show the highest number of active members.
- Silver memberships have the highest expired count, indicating possible retention issues.
- Monthly client activity peaks in November, suggesting seasonal variation or campaign impact.
- Revenue significantly exceeds expense, resulting in a strong overall profit trend.
